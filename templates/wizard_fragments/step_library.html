Library Selection
Choose your Plex music library and set the path where music files live on disk.
Select the library section that contains your music.
The absolute path where Plex finds your music files. This must exactly match what Plex reports — even a trailing slash difference will break writes.

Example: if Plex shows a track path as /mnt/media/Music/Artist/Album/01.mp3, enter /mnt/media/Music.
Plex-Sourced Lyrics

Plex Pass can fetch lyrics from providers like TIDAL and LyricFind. These are stored inside Plex (not as .lrc files). You can treat them as already covered so this tool skips re-fetching those tracks.

These can be changed at any time in Settings.
